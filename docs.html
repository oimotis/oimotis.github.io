MWeb 2.0.7 submited!

2016/7/27  posted in Tutorials  Tips & Tricks  Releases 

MWeb 2.0.7 Update:

  • Add Paste Markdown from HTML, Menu: Edit - Paste Markdown from HTML.
  • Add CMD + O support open all text document, not only .md file.
