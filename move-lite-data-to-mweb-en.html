Move the MWeb Lite Library's Data to MWeb

2015/07/31  posted in Tips & Tricks  Tutorials 

When you MWeb Lite's Library documents is text-only, you can use Export to Markdown and Input Files to Library Export/Input MWeb Lite Library's documents to MWeb.

If you MWeb Lite Library has a lot of photos and attachments, you can move all the MWeb Lite Library Data to MWeb. See the below for detail.

1. Open MWeb Lite Document Library Folder

Right click the MWeb Lite's Titlebar, selected MWeb and click, will open Finder like below:


Please notice the LocalData folder.

2. Open MWeb Document Library Folder

Right click the MWeb's Titlebar, selected MWeb and click, will open Finder like below:


Please notice the LocalData folder.

3. Move the MWeb Lite's Data to MWeb

Notice: Please backup the MWeb's LocalData folder first!

  1. Quit MWeb Lite and MWeb, notice, is CMD + Q to quit.
  2. Move the MWeb's LocalData folder to the Trash, Copy the MWeb Lite's LocalData folder to MWeb.
  3. Reopen MWeb.
  4. Finished!!
