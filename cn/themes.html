自定义主题

要自定义主题，可以参考帮助页面中的静态网站部分。也可以参考自带的几个主题，非常简单，一看即懂。

也可以付费请美女设计师 庫倪Qoli 定制一个。MWeb 2.x 版的工具栏图标就是 庫倪Qoli 重新设计的。

如果你 MWeb 制作了一个主题，想放在这个页面上分享给大家使用，欢迎联系我！邮件：coderforart+233@gmail.com、微博：@oulvhai、Twitter：@oulvhai

如何使用

把主题文件夹放在 “MWeb 静态网站主题文件夹” 内即可，可以在选择主题模板那里点 “编辑” 按钮进入 “MWeb 静态网站主题文件夹”。

如果你用 git，直接使用以下命令即可：

cd ~/Library/Containers/com.coderforart.MWeb3/Data/Documents/themes/SiteThemes
git clone '主题 git clone 网址'

注意：2.x 版的主题文件夹在：~/Library/Containers/com.coderforart.MWeb/Data/Documents/themes/SiteThemes
