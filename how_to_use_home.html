MWeb Q&A

Can't see MWeb folder in iCloud Drive on Mac

You can try to go to iOS system settings - iCloud, close iCloud Drive and open again.

If still not work, please wait for Dropbox and WebDAV support in version 2.3.x

Use MWeb for Mac Library in iOS you should know

MWeb for Mac Library in iOS only support view, edit, new document. You can find the new document in Uncategorized.
